# TestReady Web

This repository hosts the public GitHub Pages build of the TestReady web app.

The app itself is static. After opening the site, enter the private GitHub repository settings and a fine-grained personal access token so the app can read tests and write attempt JSON files.

Notes:
- GitHub token and unfinished drafts are stored in the browser local storage on each device.
- Completed attempts are saved back into your private repository, so they are available from anywhere once you sign in on that device.
